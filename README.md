# ⚡ One-Shot Workflow Generator

**Describe a workflow in plain English and watch it come to life!**

Built by Sam AI as part of the daily AI trend app series.

## What It Does

Turn natural language descriptions into visual workflow diagrams:
- "automate my daily newsletter curation" → 3-step workflow graph
- "process customer feedback emails" → automated pipeline
- "scrape news and post to Twitter" → full automation flow

Powered by Grok API with visual flowchart rendering using Cytoscape.js.

## Features

- 🤖 AI-powered workflow generation (Grok 4.1)
- 📊 Interactive flowchart visualization
- 🎨 Neobrutalist design (thick borders, bold colors)
- 📱 Mobile-responsive
- ⚡ Single-file HTML (no build step)
